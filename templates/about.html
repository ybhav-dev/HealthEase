About HealthEase

HealthEase is a basic health awareness web application that helps users check symptoms, read about common health issues, view simple health tips, and access emergency information. It is designed for learning and awareness and should not be used for medical diagnosis.

Project Overview

The main aim of HealthEase is to provide a single, clean interface where a user can:

  • Describe symptoms to a rule-based chatbot.
  • Explore diseases & remedies section.
  • Read health tips for daily lifestyle improvement.
  • Access emergency features & hospital location map.
  • Use tools like BMI calculator & future trackers.

Objectives

  • To create an easy-to-use health awareness platform.
  • To practice full-stack development with Flask.
  • To organise basic health information in UI-friendly format.
  • To demonstrate integration of tools & Google Maps.

Key Features

  • Landing page with quick feature access.
  • Basic symptom-based chatbot system.
  • Diseases & Remedies section.
  • Health Tips dashboard.
  • Emergency map & first-aid cards.
  • BMI Calculator & placeholder tools for future.

Technology Stack

  • Frontend: HTML5, CSS3, JavaScript
  • Backend: Python Flask
  • Templating: Jinja2
  • Data (planned): JSON for symptoms/diseases/tips
  • Other: Google Maps integration

Disclaimer

HealthEase is a minor project created for educational purposes only. The information shown is general and should not replace professional medical advice.

For serious health concerns, users must consult qualified healthcare professionals.

Future Scope

  • Enhance chatbot using NLP & external API.
  • Store user history (with privacy control).
  • Implement complete tools like Mood & Sleep Tracker.
  • Make multilingual versions for accessibility.
